# VuePress 链接语法参考

## 基本规则

1. **全部使用相对路径**，禁止硬编码站点绝对路径
2. 路径含空格或中文时，**必须**用 `<>` 包裹路径部分

## 语法示例

### 简单链接（路径无空格）
```markdown
[文件](./filename.md)
[目录](../parent/)
```

### 路径含空格（数字前缀目录）
```markdown
[插件开发](<../3. 插件开发/README.md>)
[快速开始](<../3. 插件开发/1. 快速开始.md>)
```

### 锚点
锚点跟在 `>` 后面写，**不**写在 `<>` 内：
```markdown
[标题](<file.md>#section)
[基类参考](<../5. 插件系统/1. 基类.md>#3-pluginmanifest)
```

### 纯 URL
```markdown
<https://github.com/ncatbot/NcatBot>
```

## 解析规则（供检查脚本使用）

1. 匹配 `[text](<path>)` → 提取 `path`（剥离 `<>`）
2. 匹配 `[text](path)` → 提取 `path`
3. 如果 `path` 以 `http://` 或 `https://` 开头 → 外部链接，跳过
4. 如果 `path` 含 `#` → 取 `#` 前面部分作为文件路径
5. 基于当前文件所在目录解析相对路径
6. 验证解析后的绝对路径是否存在
